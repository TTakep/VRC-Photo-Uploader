# VRC Photo Uploader for Discord

VRChatなどで撮影した高解像度（4K/8K）の写真を、**PNG形式のまま画質を劣化させることなく** DiscordにアップロードするためのWebツールです。

ブラウザだけで動作し、サーバーに画像を送信しないため、プライバシーを守りながら高速に動作します。

## ✨ 特徴

* **画質最優先のリサイズ処理**:
  * ファイルサイズがDiscordの制限（デフォルト8MB）を超えている場合のみ処理を行います。
  * JPEG変換や圧縮率の変更は行わず、**PNG形式を維持したまま解像度（ピクセル数）だけを縮小**して容量を収めます。これにより、ブロックノイズのない綺麗な画像を送信できます。
* **完全クライアントサイド動作**:
  * 画像処理はすべてお使いのブラウザ（JavaScript）内で行われます。
  * 外部サーバーに写真がアップロードされることはありません。
* **Discord Webhook対応**:
  * Webhookを使用するため、ご自身のサーバーや特定のチャンネルに直接送信できます。
* **レートリミット対策**:
  * Discord APIの制限を考慮し、複数枚の送信時に自動でウェイト（待機時間）を挟みます。
* **設定の自動保存**:
  * Webhook URLや設定値はブラウザに保存され、次回以降は入力不要です。

## 🚀 使い方

### 1. 準備（初回のみ）
1. 写真を投稿したいDiscordサーバーのチャンネル設定を開きます。
2. 「連携サービス」→「ウェブフック」を作成します。
3. 「ウェブフックURLをコピー」をクリックします。

### 2. アップロード
1. このツールを開きます。
2. **設定** エリアにある「Discord Webhook URL」に、コピーしたURLを貼り付けます。
3. **ドラッグ＆ドロップエリア** に写真を放り込みます（複数選択可）。
4. 自動的にサイズチェックとリサイズが行われ、Discordに送信されます。

## ⚙️ 技術仕様

* **Frontend**: HTML5, JavaScript (Vanilla)
* **Styling**: Bootstrap 5 (CDN)
* **Image Processing**: HTML5 Canvas API
* **Data Storage**: LocalStorage (Webhook URL保存用)

## 🔒 セキュリティとプライバシー

* **Webhook URL**: ブラウザの `localStorage` に保存されます。外部のデータベース等には送信されません。
* **画像データ**: ブラウザ内で処理され、直接DiscordのAPIへ送信されます。開発者やホスティング元のサーバーを経由することはありません。

## ⚠️ 注意事項

* **共有PCでの利用**: Webhook URLがブラウザに保存される仕様上、不特定多数が使う共有PCやネットカフェ等での利用後は、ブラウザのキャッシュ/データを削除することを推奨します。
* **Discordの制限**: Webhookの仕様上、極端に短時間で大量の画像を送信すると一時的に制限（429エラー）がかかる場合があります。本ツールは自動で待機・再送を行いますが、常識的な範囲でご使用ください。
